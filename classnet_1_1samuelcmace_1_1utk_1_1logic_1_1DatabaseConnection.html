◆ finalize()

void net.samuelcmace.utk.logic.DatabaseConnection.finalize ( )
protected

Destructor called by the garbage collector to close the database connection on program exit.

Although destructors are no longer needed in most java programming, this context requires that the database connection will remain open for the entire lifespan of the object, but also closed when the object is destroyed.
